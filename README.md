# House Price Prediction using Linear Regression

This project demonstrates how **Linear Regression** can be used to predict house prices based on housing features.  
It covers the complete workflow of a basic machine learning regression problem.

---

## 📌 Project Description

The goal of this project is to build a regression model that predicts house prices using historical housing data.  
Linear Regression is applied to understand the relationship between house size and price.

This project is suitable for beginners who want hands-on experience with regression models and data preprocessing.

---

## 🛠️ Tools and Technologies

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

---

## 📂 Dataset

The dataset contains housing-related information, including:
- Living area of the house (`sqft_living`)
- House price (`price`)

### Data Cleaning:
- Removed invalid records where the living area was zero
- Selected relevant features for regression analysis

---

## ⚙️ Approach

1. Load and explore the dataset  
2. Clean the data and handle invalid values  
3. Split the dataset into training and testing sets  
4. Train a Linear Regression model  
5. Make predictions on test data  
6. Evaluate model performance using standard regression metrics

---

## 📊 Evaluation Metrics

- **R² Score** – Measures how well the model explains variance in the data  
- **Mean Squared Error (MSE)** – Measures the average squared prediction error
